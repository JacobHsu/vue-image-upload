## node upload api

`$ cd node_upload_api`  
`$ yarn init -y`  
`$ yarn add express nodemon`  
`$ yarn add multer`  

Usage `$ yarn start`

[Multer](https://github.com/expressjs/multer) is a node.js middleware for handling multipart/form-data , which is primarily used for uploading files.
